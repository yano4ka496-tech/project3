# План реализации: Конвертация PDF в MBTiles и интеграция с приложением SafePlant

## Общая информация
Проект SafePlant — мобильное приложение для обеспечения безопасности на промышленных объектах. Приложение использует офлайн-архитектуру с векторными картами, системой допусков и обучением.

## Архитектура
- **Модульный монолит** с паттерном MVVM и Clean Architecture
- **Слои**: presentation, domain, data
- **Технологии**: Android SDK, Jetpack Compose, Room, MapLibre GL Native
- **Безопасность**: шифрование через Android Keystore, проверка root-доступа

## Критерии приемки
- [ ] MBTiles корректно отображается в MapLibre GL
- [ ] Масштабирование от 10 км до 10 м работает без артефактов
- [ ] Скрипт конвертации PDF в MBTiles документирован
- [ ] Размер файла < 200 МБ с автоматической оптимизацией
- [ ] QR-коды привязываются к объектам по objectId
- [ ] Обработка ошибок QR-сканирования реализована
- [ ] Сброс допуска при обновлении версии работает
- [ ] Обнаружение root-доступа реализовано
- [ ] Доступ к квизу без ограничений при отсутствии допуска

## Проверка
- Конвертация PDF в MBTiles: `python3 scripts/convert_pdf_to_mbtiles.py tests/data/pdf/test.pdf tests/data/mbtiles/test.mbtiles`
- Валидация MBTiles: `python3 scripts/validate_mbtiles.py tests/data/mbtiles/test.mbtiles`
- Проверка зависимостей: `bash scripts/check_pdf_conversion.sh`
- Unit-тесты: `./gradlew test`
- Интеграционные тесты: `./gradlew connectedAndroidTest`
- Проверка линтинга: `./gradlew ktlintCheck`
- Проверка детекта: `./gradlew detekt`
- Запуск всех тестов: `bash scripts/run_all_tests.sh`
- Финальная проверка: `bash scripts/run_final_checks.sh`
