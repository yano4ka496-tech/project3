# Specification: Логика квиза

## Context
Необходимость проверки знаний пользователя по технике безопасности. Квиз должен случайным образом выбирать 10 вопросов и подсчитывать баллы.

## Functional Requirements
- FR-01: Загружать вопросы из Room (QuizQuestion)
- FR-02: При старте случайным образом выбирать 10 вопросов (детерминированный seed)
- FR-03: Реализовать тип вопросов: одиночный выбор
- FR-04: Подсчитывать правильные ответы
- FR-05: Use case возвращает успех (≥8) или неудачу

## Acceptance Criteria
- [ ] Вопросы выбираются случайным образом
- [ ] Подсчёт баллов корректен
- [ ] Use case покрыт unit-тестами
- [ ] При пересдаче вопросы меняются
