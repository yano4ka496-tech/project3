# Clarifications: 02-01::navigation-setup

## Question 1
Как обрабатывать навигацию, если QR-код содержит некорректный zoneId (например, несуществующий в базе данных)?

*Context:* Это влияет на обработку ошибок и пользовательский опыт при сканировании QR-кодов

**Options:**
- A) Показать экран ошибки с сообщением 'Зона не найдена'
- B) Перенаправить на карту с координатами по умолчанию (например, центр объекта)
- C) Игнорировать и остаться на экране QR-сканера
- D) Не обрабатывать - считать это ошибкой приложения

**Answer:**
Показать экран ошибки с сообщением 'Зона не найдена'
