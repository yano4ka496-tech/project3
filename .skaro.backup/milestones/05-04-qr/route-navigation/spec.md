# Specification: Навигация по маршрутам

## Context
Необходимость отображения предустановленных маршрутов на карте. Маршруты должны проверяться на наличие допуска.

## Functional Requirements
- FR-01: Реализовать выбор цели из списка
- FR-02: Подсвечивать соответствующий маршрут из routes.geojson
- FR-03: Отображать направление движения
- FR-04: Блокировать маршрут при отсутствии допуска через опасную зону
- FR-05: Реализовать сброс маршрута по кнопке

## Acceptance Criteria
- [ ] Маршрут подсвечивается на карте
- [ ] Блокировка при отсутствии допуска работает
- [ ] Направление движения отображается корректно
- [ ] Сброс маршрута работает без сбоев
