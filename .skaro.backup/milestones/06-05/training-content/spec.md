# Specification: Обучающий контент

## Context
Необходимость отображения инструкций по технике безопасности. Контент должен быть организован по разделам с возможностью поиска.

## Functional Requirements
- FR-01: Загружать инструкции из assets (JSON/Markdown)
- FR-02: Реализовать разделы: общие правила, ЧС, штрафы, знаки
- FR-03: Реализовать поиск по ключевым словам
- FR-04: Реализовать закладки (локальное сохранение)
- FR-05: Адаптивный дизайн для разных размеров экрана

## Acceptance Criteria
- [ ] Контент корректно отображается
- [ ] Поиск работает с учетом регистра
- [ ] Закладки сохраняются между сессиями
- [ ] Интерфейс адаптивен
