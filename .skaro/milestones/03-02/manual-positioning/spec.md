# Specification: Ручное позиционирование

## Context
Необходимость возможности ручного выбора позиции пользователя, когда QR-сканирование невозможно. Позиция должна сохраняться между сессиями.

## Functional Requirements
- FR-01: Реализовать кнопку "Выбрать место на карте"
- FR-02: Реализовать режим выбора точки (тап по карте)
- FR-03: Сохранять координаты в EncryptedSharedPreferences
- FR-04: Отображать маркер в выбранной точке
- FR-05: Обновлять позицию при новом выборе

## Acceptance Criteria
- [ ] Ручной выбор позиции работает корректно
- [ ] Позиция сохраняется после перезапуска приложения
- [ ] Маркер отображается на карте в выбранной точке
- [ ] Позиция обновляется при новом выборе
