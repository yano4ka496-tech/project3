- Версия базы данных должна соответствовать последней миграции (Migration3To4)
- DatabaseKeyManager должен использоваться для создания зашифрованной базы данных
- Обработка потери ключа шифрования необходима для восстановления работы приложения после сброса устройства
- Исправление ошибки компиляции необходимо для корректной работы репозитория
