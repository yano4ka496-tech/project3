# Specification: Позиционирование пользователя

## Context
Необходимость отображения текущей позиции пользователя на карте. Позиция должна обновляться при сканировании QR или ручном выборе.

## Functional Requirements
- FR-01: Сохранять координаты в EncryptedSharedPreferences
- FR-02: При запуске читать последние координаты и отображать маркер
- FR-03: Показывать кнопку "Отсканируйте QR или выберите место" при отсутствии позиции
- FR-04: Обновлять позицию при новом сканировании или выборе
- FR-05: Реализовать анимацию перемещения камеры к позиции

## Acceptance Criteria
- [ ] Позиция сохраняется между сессиями
- [ ] Маркер отображается на карте в текущей позиции
- [ ] Кнопка показывается при отсутствии позиции
- [ ] Анимация перемещения камеры работает корректно
