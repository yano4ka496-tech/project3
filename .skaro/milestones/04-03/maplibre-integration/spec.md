# Specification: Интеграция MapLibre

## Context
Необходимость отображения векторной карты с поддержкой масштабирования. Карта должна отображаться только при наличии действующего допуска.

## Functional Requirements
- FR-01: Реализовать MapView в Compose
- FR-02: Загружать MBTiles из assets
- FR-03: Реализовать управление камерой (перемещение, масштаб)
- FR-04: Оптимизировать производительность (60 FPS)
- FR-05: Показывать заглушку при отсутствии допуска
- FR-06: Реализовать проверку наличия допуска перед отображением

## Acceptance Criteria
- [ ] Карта отображается при наличии допуска
- [ ] Заглушка показывается при отсутствии допуска
- [ ] Масштабирование от 10 км до 10 м работает без лагов
- [ ] Производительность соответствует 60 FPS
