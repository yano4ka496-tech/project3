# Specification: Слои карты

## Context
Необходимость отображения различных слоёв на карте (опасные зоны, маршруты, эвакуация, СИЗ). Слои должны быть интерактивными.

## Functional Requirements
- FR-01: Загружать GeoJSON из assets
- FR-02: Реализовать слой опасных зон (красные полигоны)
- FR-03: Реализовать слой предустановленных маршрутов (синие линии)
- FR-04: Реализовать слой эвакуации (зелёные линии) с переключением
- FR-05: Реализовать слой точек СИЗ (иконки)
- FR-06: Реализовать всплывающие подсказки при тапе на опасную зону
- FR-07: Реализовать механизм переключения слоя эвакуации через кнопку на карте
- FR-08: Реализовать проверку наличия действующего допуска при отображении слоя эвакуации
- FR-09: Реализовать появление слоёв только при достижении определенного масштаба

## Стили отображения слоёв
- Опасные зоны: красные полигоны с заданной прозрачностью
- Маршруты: синие линии с заданной толщиной
- Эвакуация: зелёные линии с заданной толщиной
- Точки СИЗ: желтые иконки

## Обработка ошибок
- FR-10: При отсутствии или повреждении GeoJSON-файлов отображать карту без соответствующего слоя и уведомлять пользователя

## Взаимодействие с элементами
- FR-11: При нажатии на элемент слоя (опасная зона, точка СИЗ) показывать всплывающее окно с информацией об объекте (название, описание)

## Acceptance Criteria
- [ ] Все слои корректно отображаются на карте
- [ ] Кнопка переключения эвакуации работает
- [ ] Подсказки появляются при тапе на элементы
- [ ] Слои не конфликтуют визуально
- [ ] Слои появляются только при достижении определенного масштаба
- [ ] При отсутствии или повреждении GeoJSON-файлов отображается уведомление
- [ ] При попытке отображения слоя эвакуации без действующего допуска показывается уведомление
- [ ] Всплывающие окна содержат корректную информацию об объектах
- [ ] Стили слоей соответствуют требованиям (цвета, толщина линии, прозрачность)
