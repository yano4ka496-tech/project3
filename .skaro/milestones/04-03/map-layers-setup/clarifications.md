# Clarifications: 04-03::map-layers-setup

## Question 1
Какой стиль отображения должен использоваться для каждого слоя карты (цвет, толщина линии, прозрачность)?

*Context:* Необходимо определить визуальные параметры для каждого слоя карты для корректного отображения и избежания визуальных конфликтов.

**Options:**
- A) Использовать стандартные стили: красный для опасных зон, синий для маршрутов, зеленый для эвакуации, желтый для СИЗ
- B) Разрешить пользователю настраивать стили через настройки приложения
- C) Использовать стили, определенные в GeoJSON-файлах
- D) Не задавать стили явно, полагаться на стили по умолчанию MapLibre

**Answer:**
Использовать стандартные стили: красный для опасных зон, синий для маршрутов, зеленый для эвакуации, желтый для СИЗ

- C) Перемещать камеру к элементу и показывать его в центре

## Question 3
Как обрабатывать ситуацию, когда GeoJSON-файлы отсутствуют или повреждены?

*Context:* Необходимо предусмотреть обработку ошибок при загрузке слоев карты.

**Options:**
- A) Показывать карту без соответствующего слоя и уведомлять пользователя
- B) Загружать пустые слои и отображать карту без элементов
- C) Блокировать отображение карты до исправления проблемы

## Question 4
Какой механизм переключения слоя эвакуации должен быть реализован?

*Context:* Необходимо определить способ включения/отключения слоя эвакуации на карте.

**Options:**
- A) Переключатель в виде кнопки на карте
- B) Переключатель в настройках приложения
- C) Жестами на карте (например, двойное нажатие)

## Question 6
Какой уровень детализации должен быть у каждого слоя при разных масштабах?

*Context:* Необходимо определить, как слои должны отображаться при разных уровнях масштабирования карты.

**Options:**
- A) Все слои отображаются на всех масштабах
- B) Слои появляются только при достижении определенного масштаба
- C) Слои меняют стиль или количество элементов в зависимости от масштаба
