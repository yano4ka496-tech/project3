# AI_NOTES — Stage 3: Интеграция слоев карты с экраном и обработка ошибок

## What was done
- Обновлен MapViewModel для управления состоянием слоев карты
- Реализована проверка доступа для слоя эвакуации
- Добавлена обработка ошибок загрузки GeoJSON-файлов
- Обновлен MapScreen для отображения слоев и кнопки переключения эвакуации
- Добавлены уведомления об ошибках загрузки слоев
- Реализована логика видимости слоев в зависимости от масштаба
- Обновлены MapView, MapController и MapRenderer для работы со слоями
- Добавлены UI-тесты для MapScreen

## Why this approach
- Использована существующая архитектура MVVM с разделением ответственности
- Состояние слоев управляется через ViewModel, что позволяет легко обновлять UI
- Проверка доступа для слоя эвакуации интегрирована в бизнес-логику
- Обработка ошибок централизована в ViewModel с уведомлениями в UI
- Кнопка переключения эвакуации находится в удобном месте (правый верхний угол)
- Видимость слоев обновляется при изменении масштаба для оптимизации производительности

## Risks and limitations
- В текущей реализации MapView является заглушкой, в реальном приложении потребуется интеграция с MapLibre GL Native
- Обработка ошибок загрузки GeoJSON базовая, в реальном приложении потребуется более детальная обработка
- Логика видимости слоев в зависимости от масштаба упрощена, в реальном приложении может потребоваться более сложная логика
- Тесты используют моки, в реальном приложении потребуется интеграционное тестирование с реальной картой

## How to verify
1. Проверить сборку проекта: `./gradlew :feature-map:assembleDebug`
2. Запустить unit-тесты для MapViewModel: `./gradlew :feature-map:testDebugUnitTest --tests "com.safeplant.feature.map.MapViewModelTest"`
3. Запустить UI-тесты для MapScreen: `./gradlew :feature-map:testDebugUnitTest --tests "com.safeplant.feature.map.MapScreenTest"`
4. Проверить, что кнопка переключения эвакуации работает корректно
5. Убедиться, что при отсутствии доступа слой эвакуации не отображается
6. Проверить, что уведомления об ошибках появляются при проблемах загрузки слоев
