# Specification: Видеоплеер

## Context
Необходимость воспроизведения обучающих видео (до 50 МБ каждое). Видео должны копироваться из assets при первом запуске.

## Functional Requirements
- FR-01: При первом запуске копировать видео из assets во внутреннее хранилище
- FR-02: Реализовать ExoPlayer для воспроизведения локальных файлов
- FR-03: Реализовать UI: плейлист, пауза/воспроизведение, прогресс-бар
- FR-04: Обрабатывать ошибки: поврежденный файл, не скопирован
- FR-05: Проверять общий размер видео (<150 МБ)

## Acceptance Criteria
- [ ] Видео воспроизводятся без лагов
- [ ] Копирование происходит один раз
- [ ] UI плеера интуитивно понятен
- [ ] Обработка ошибок работает корректно
