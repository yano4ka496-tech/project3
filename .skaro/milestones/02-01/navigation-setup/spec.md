# Specification: Настройка навигации

## Context
Необходимость реализации навигации между экранами приложения и поддержки deep links из QR-сканера. Навигация должна быть интегрирована с Hilt.

## Functional Requirements
- FR-01: Создать NavHost с экранами: карта, квиз, обучение, профиль, QR-сканер
- FR-02: Реализовать deep link: safeplant://map?zoneId={zoneId}
- FR-03: Обеспечить переход из QR-сканера на карту с zoneId
- FR-04: Интегрировать Hilt с NavController
- FR-05: Реализовать обработку ошибок навигации
- FR-06: Показывать экран ошибки 'Зона не найдена' при попытке навигации с некорректным zoneId
- FR-07: Показывать диалог с предложением пройти квиз при попытке навигации на карту без действующего допуска
- FR-08: При запуске через deep link, если пользователь уже находится на другом экране, немедленно переключиться на карту с указанной зоной
- FR-09: При запуске через deep link без указания zoneId, показывать карту с центром по умолчанию
- FR-10: Показывать экран ошибки 'Зона не найдена' при запуске через deep link, если QR-код для указанной зоны не найден в базе
- FR-11: Показывать экран ошибки 'Доступ запрещен' при запуске через deep link, если пользователь не имеет доступа к указанной зоне

## Acceptance Criteria
- [ ] Deep link корректно открывает карту с нужной зоной
- [ ] Навигация между экранами работает без сбоев
- [ ] Hilt предоставляет зависимости для всех экранов
- [ ] Обработка ошибок навигации реализована
- [ ] При попытке навигации с некорректным zoneId показывается экран ошибки 'Зона не найдена'
- [ ] При попытке навигации на карту без действующего допуска показывается диалог с предложением пройти квиз
- [ ] При запуске через deep link, если пользователь уже находится на другом экране, происходит немедленный переход на карту с указанной зоной
- [ ] При запуске через deep link без указания zoneId показывается карта с центром по умолчанию
- [ ] При запуске через deep link с некорректным zoneId показывается экран ошибки 'Зона не найдена'
- [ ] При запуске через deep link без доступа к зоне показывается экран ошибки 'Доступ запрещен'
