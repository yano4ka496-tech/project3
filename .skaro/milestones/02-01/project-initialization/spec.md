# Specification: Инициализация проекта

## Context
Необходимость создания модульной структуры проекта с правильными зависимостями и настройками CI/CD. Проект должен соответствовать требованиям конституции (Kotlin 1.9+, Compose, Room, Hilt).

## Functional Requirements
- FR-01: Создать модули: app, core-navigation, core-database, core-storage, core-security, core-mapping, feature-quiz, feature-map, feature-training, feature-qr, feature-profile
- FR-02: Настроить Gradle: Kotlin 1.9+, Compose BOM 2024.10+, Room 2.8+, Hilt 2.51+, MapLibre GL Native
- FR-03: Настроить Hilt: AppModule, DatabaseModule, RepositoryModule
- FR-04: Настроить ktlint 1.8+ и Detekt (конфиг по умолчанию + Android)
- FR-05: Настроить Gradle tasks для копирования assets (MBTiles, GeoJSON, CSV, видео)
- FR-06: Включить все ресурсы (MBTiles, GeoJSON, видео) в APK как assets
- FR-07: Реализовать обработку некорректных QR-кодов: показ сообщения об ошибке и предложение повторить сканирование
- FR-08: Реализовать логику викторины: оценка результатов и возможность повторения до достижения проходного балла (8/10)
- FR-09: Реализовать проверку версии приложения при запуске для сброса цифрового пропуска при изменении версии

## Database Structure
- DB-01: Использовать Room с автоматическими миграциями
- DB-02: Основные сущности: User, QuizResult, AccessPass, MapObject, TrainingVideo
- DB-03: Все данные в базе должны быть зашифрованы через Android Keystore

## Acceptance Criteria
- [ ] Проект компилируется без ошибок
- [ ] ./gradlew ktlintCheck detekt проходит без ошибок
- [ ] Assets корректно копируются в build
- [ ] Все модули подключены через Hilt
- [ ] Приложение корректно обрабатывает некорректные QR-коды
- [ ] Пользователь может пройти викторину до достижения проходного балла
- [ ] Цифровой пропуск сбрасывается при изменении версии приложения
- [ ] База данных корректно мигрируется при обновлении версии
- [ ] Все ресурсы доступны в runtime через assets

## Open Questions (Resolved)

- [ ] Как оценивать викторину? → Позволить повторить до 8/10 баллов (FR-08)
