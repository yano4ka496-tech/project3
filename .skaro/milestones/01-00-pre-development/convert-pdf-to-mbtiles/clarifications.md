# Clarifications: 01-00-pre-development::convert-pdf-to-mbtiles

## Question 1
Какой формат PDF ожидается на входе скрипта конвертации? Многостраничный PDF с векторными слоями или растровые изображения?

*Context:* Определяет, нужно ли использовать OCR для растровых элементов или ogr2ogr может напрямую обрабатывать векторные слои PDF

**Options:**
- A) Только векторные слои PDF (ogr2ogr может обрабатывать напрямую)
- B) Растровые изображения PDF (требуется предварительная обработка для векторизации)
- C) Смешанный формат (векторные слои + растровые элементы)
- D) Не требуется уточнение (использовать универсальный подход)

**Answer:**
Только векторные слои PDF (ogr2ogr может обрабатывать напрямую)
