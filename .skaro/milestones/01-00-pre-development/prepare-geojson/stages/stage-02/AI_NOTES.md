# AI_NOTES — Stage 2: Реализация GeoJSON файлов с данными

## What was done
- Заполнены все четыре GeoJSON файла реальными данными в соответствии с требованиями:
  - `hazards.geojson`: добавлены три полигона опасных зон (химические вещества, высокое напряжение, горючие материалы)
  - `routes.geojson`: добавлены три линии маршрутов (основной, альтернативный, к зоне СИЗ)
  - `ppe_points.geojson`: добавлены четыре точки СИЗ (каски, респираторы, защитная одежда, перчатки)
  - `evacuation.geojson`: добавлены три пути эвакуации (основной, запасной, через восточный коридор)
- Все координаты соответствуют системе WGS84 (EPSG:4326) в районе Москвы
- Каждый объект содержит свойства для стилизации (name, description, color, category)
- Создан скрипт `scripts/validate_geojson.py` для валидации GeoJSON файлов

## Why this approach
- Использованы реальные координаты в районе Москвы для демонстрации функциональности
- Каждый тип геометрии (Polygon, LineString, Point) соответствует требованиям
- Добавлены дополнительные свойства (category) для возможной фильтрации и стилизации
- Скрипт валидации проверяет все требования к GeoJSON форматам
- Цветовая схема соответствует требованиям (красный для опасных зон, синий для маршрутов, зеленый для СИЗ и эвакуации)

## Risks and limitations
- Координаты являются тестовыми и должны быть заменены на реальные данные объекта
- Размер файлов небольшой, но при увеличении количества объектов может потребоваться оптимизация
- Скрипт валидации не проверяет корректность самих координат (например, находятся ли они в пределах объекта)
- Нет проверки на пересечение объектов или перекрытие слоев

## Invariant compliance
- [ ] Все файлы содержат валидные данные в формате GeoJSON — соблюдено
- [ ] Координаты соответствуют системе WGS84 (EPSG:4326) — соблюдено
- [ ] Опасные зоны представлены как полигоны — соблюдено
- [ ] Маршруты и пути эвакуации представлены как LineString — соблюдено
- [ ] Точки СИЗ представлены как Point — соблюдено
- [ ] Каждый файл содержит свойства для стилизации (цвет, название) — соблюдено

## How to verify
1. Запустить скрипт валидации: `python scripts/validate_geojson.py app/src/main/assets/geojson/hazards.geojson app/src/main/assets/geojson/routes.geojson app/src/main/assets/geojson/ppe_points.geojson app/src/main/assets/geojson/evacuation.geojson`
2. Проверить, что все файлы прошли валидацию без ошибок
3. Открыть файлы в geojson.io для визуальной проверки
4. Убедиться, что координаты соответствуют ожидаемым местоположениям
