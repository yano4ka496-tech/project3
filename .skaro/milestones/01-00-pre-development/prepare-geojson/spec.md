# Specification: Подготовка GeoJSON-файлов

## Context
Необходимость создания векторных слоёв для отображения опасных зон, маршрутов, точек СИЗ и путей эвакуации. Геоданные должны соответствовать координатной системе MBTiles.

## Functional Requirements
- FR-01: `hazards.geojson` – полигоны опасных зон (красный цвет)
- FR-02: `routes.geojson` – линии предустановленных маршрутов (синий цвет)
- FR-03: `ppe_points.geojson` – точки СИЗ (иконки)
- FR-04: `evacuation.geojson` – пути эвакуации (зелёный цвет)
- FR-05: Разместить в `app/src/main/assets/geojson/`
- FR-06: Проверить соответствие координат системе WGS84

## Acceptance Criteria
- [ ] Все GeoJSON файлы валидны (проверка через geojson.io)
- [ ] Координаты соответствуют MBTiles
- [ ] Слои визуально корректно накладываются на карту
