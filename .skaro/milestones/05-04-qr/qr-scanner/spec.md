# Specification: QR-сканер

## Context
Необходимость сканирования QR-кодов для позиционирования пользователя. Сканер должен обрабатывать ошибки и некорректные форматы.

## Functional Requirements
- FR-01: Реализовать CameraX для сканирования QR
- FR-02: Передавать распознанный текст в парсер
- FR-03: Искать координаты в QrCodeMapping (Room)
- FR-04: Переходить на карту с центром на координатах
- FR-05: Обрабатывать ошибки: неверный формат, неизвестный id, недоступная камера
- FR-06: Показывать сообщение 'Объект не найден' при сканировании QR-кода с неизвестным objectId
- FR-07: Показывать предупреждение о нахождении в опасной зоне при сканировании QR-кода в опасной зоне
- FR-08: Обрабатывать только QR-коды формата 'objectId|name', где objectId - латинские буквы и цифры, name - любые символы
- FR-09: Показывать сообщение 'Камера не поддерживается' и предлагать выбрать позицию вручную при отсутствии камеры
- FR-10: Показывать предупреждение о необходимости обновления допуска при сканировании QR-кода с истекшим сроком действия
- FR-11: Автоматически включать вспышку и показывать сообщение 'Включите вспышку для лучшего сканирования' в условиях низкой освещенности

## Acceptance Criteria
- [ ] QR сканируется и распознается корректно
- [ ] Карта открывается с правильной позицией
- [ ] Ошибки показывают информативные сообщения
- [ ] Сканер работает в условиях низкой освещенности
- [ ] При сканировании QR-кода с неизвестным objectId показывается сообщение 'Объект не найден'
- [ ] При сканировании QR-кода в опасной зоне позиционирование разрешается, но показывается предупреждение
- [ ] При сканировании QR-кода с истекшим сроком действия допуска позиционирование разрешается, но показывается предупреждение
- [ ] При отсутствии камеры на устройстве показывается сообщение 'Камера не поддерживается' и предлагается выбрать позицию вручную
- [ ] В условиях низкой освещенности автоматически включается вспышка и показывается сообщение 'Включите вспышку для лучшего сканирования'
- [ ] Приложение корректно обрабатывает только QR-коды формата 'objectId|name'
